# portfolio

A minimal personal portfolio site hosting an undergraduate thesis — plain HTML/CSS, no framework, served by GitHub Pages.

**Live site**: https://natsunooda.github.io/portfolio/

## What's where

- `index.html` / `style.css` — the whole site
- `thesis.pdf` — the thesis itself
- `CLAUDE.md` — structure and deploy notes (canonical reference for contributors/tools)

## Local preview

```bash
python3 -m http.server 8080
```

Then open http://localhost:8080/.
